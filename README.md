# India Census Data Analysis (Python)

This project performs exploratory data analysis on the **2011 India Census dataset** using Python. The goal is to understand demographic patterns such as population distribution, literacy rate, gender ratio, urbanization, and population density across Indian states.

## Tools and Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset

The dataset used in this project is based on the **2011 Census of India** and contains information about population, literacy, gender distribution, and rural/urban population across different states.

## Project Objectives

* Analyze total population distribution across states
* Identify the most urbanized states
* Compare literacy rates among states
* Study gender ratio patterns
* Understand population density distribution

## Data Visualizations

The project includes multiple visualizations to represent insights clearly:

* **Horizontal Bar Chart** – State-wise total population comparison
* **Pie Chart** – Urban population share of top urbanized states
* **Dot Plot** – Literacy rate comparison across states
* **Lollipop Chart** – Gender ratio by state
* **Violin Plot** – Population density distribution

## Files in This Project

* Python Project Codes.py – Python script for data analysis and visualization
* 2011-IndiaStateDistSbDist-0000.xlsx – Census dataset used for analysis

## Key Learning Outcomes

* Data cleaning and preprocessing using Pandas
* Data visualization with Matplotlib and Seaborn
* Exploratory Data Analysis (EDA)
* Understanding demographic patterns through visual insights
